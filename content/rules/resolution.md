# Resolution

The GM resolves each phase in order. For each cell, the player's score is compared against the opposition.

## Outcome Ratings

| Rating | Meaning |
|--------|---------|
| **Superior** | Dominating success — bonus effects |
| **Excellent** | Clean execution |
| **Good** | Success with minor complications |
| **Adequate** | Bare success — costs attached |
| **Poor** | Failure with consequences |
| **None** | Total failure |
| **Flatlined** | Catastrophic — crew loss possible |

The GM writes the **dispatch** — a narrative summary of the round — and sends it to all players. Your next round starts from whatever the dispatch says is true.
