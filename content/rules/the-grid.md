# The Operation Grid

The Grid is a **4×3 matrix**. Rows are phases. Columns are modes of force.

## Phases (Rows)

| Phase | Name | Description |
|-------|------|-------------|
| 1 | **Approach** | Recon, disguise, insertion |
| 2 | **Breach** | Entry, ICE, suppression |
| 3 | **Execute** | Primary objective |
| 4 | **Extract** | Escape, cover, decoy |

## Columns

- **Ghost** — Stealth, intel, net operations
- **Strike** — Force, speed, breach
- **Anchor** — Hold, presence, control

You assign crew members to cells. Each crew member has a **skill rating** and a **specialty**. The GM holds the opposition grid in secret. You never see it. You read the world and bet.
